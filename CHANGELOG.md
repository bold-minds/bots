# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-05

### Added

- **Ladder plugin** — Personal growth companion with `/climb` command
- **Enneagram-based onboarding** — Quick (5 question) or full (12 question) personality assessment with wing identification
- **7 calibration dimensions** — Accountability intensity, truth delivery, motivation anchor, reflection depth, structure preference, balance challenge, coaching style
- **Enneagram-to-defaults mapping** — Pre-set calibration for all 9 types
- **3 onboarding paths** — Full setup, quick start, or skip-to-goals
- **Goal tracking** — 7-step goal creation with external structure requirements, binary accountability, and WOOP integration
- **Boundary goals** — For users whose challenge is balancing (one area crowding others)
- **Weekly reflections** — Depth-calibrated reviews with commitment scorecards, pattern analysis, and harsh truth questions
- **Accountability skill** — Mirror, challenge, hold the line process calibrated to user's intensity setting
- **Coach skill** — 9 topic areas with 20+ published thought leaders, delivered in user's preferred coaching style
- **Seeded rationalization patterns** — 5-7 patterns per Enneagram type, validated through observation over time
- **5 quality gates** — Goal definition, profile completeness, commitment verification, reflection integrity, pattern validation
- **Safety disclaimer** — Crisis redirect with 988 lifeline reference
- **Deep dive videos** — You've Got A Type YouTube links for all 9 Enneagram types
- **Post-onboarding research** — Background Enneagram deep dive, custom author references, pattern enrichment
- **Date/time verification rule** — System checks real time before every time-sensitive action
